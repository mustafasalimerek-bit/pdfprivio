# PDFKitsy

Offline PDF tools for iOS and Android. Merge, split, compress, sign, OCR — all on-device, no cloud uploads, no subscription required.

## Stack

- **Flutter** (Dart 3.11+)
- **Riverpod** for state
- **PDFKit / pdfx / syncfusion_flutter_pdf** for PDF operations
- **Apple Vision** for on-device OCR (iOS)
- **in_app_purchase** for Pro tier (lifetime + sub options)

## Positioning

- **On-device only** — files never leave the phone
- **Honest free tier** — merge, split, compress, sign, annotate all free, unlimited
- **One-time purchase** — $19.99 Lifetime ends subscription fatigue
- **Bulletproof reliability** — handles 500+ page PDFs that other apps choke on

## Bundle

- iOS: `com.erekstudio.pdfkitsy`
- Android: `com.erekstudio.pdfkitsy`
